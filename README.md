# AI Workout Coach

A personalized workout recommendation website built with modern web standards.

## Features

- **Personalized Workouts:** Get exercise plans based on your fitness level and goals.
- **Modern UI:** Responsive design with glassmorphism and smooth animations.
- **Dark/Light Mode:** Seamlessly switch between themes with preference persistence.
- **Web Components:** Uses custom elements for encapsulated and reusable UI.
- **Lucide Icons:** Integrated beautiful iconography.

## Tech Stack

- HTML5
- CSS3 (Baseline features like Container Queries, `:has()`, Cascade Layers)
- JavaScript (ES Modules, Web Components)
- [Lucide](https://lucide.dev/) for icons
- [Google Fonts](https://fonts.google.com/) for typography

## Development

Simply open `index.html` in your browser or use a local development server.

## Deployment

This project is ready to be hosted on GitHub Pages or any static hosting service.
